O que é a tecnologia Java e por que preciso dela?

Java é uma linguagem de programação e plataforma de computação lançada pela primeira vez pela Sun Microsystems em 1995. Ela evoluiu de um começo humilde para impulsionar uma grande parte do mundo digital de hoje, fornecendo a plataforma confiável sobre a qual muitos serviços e aplicativos são construídos. Produtos e serviços digitais novos e inovadores projetados para o futuro também continuam a contar com Java.

Embora a maioria dos aplicativos Java modernos combinem o tempo de execução e o aplicativo Java, ainda existem muitos aplicativos e até mesmo alguns sites que não funcionarão a menos que você tenha um Java para desktop instalado. Java.com, este site, é destinado a consumidores que ainda podem precisar de Java para seus aplicativos de desktop - especificamente aplicativos direcionados ao Java 8. Desenvolvedores e usuários que desejam aprender programação Java devem visitar o site dev.java e usuários corporativos deve visitar oracle.com/java para obter mais informações.

A linguagem Java é fortemente tipada. Suas especificações distinguem claramente entre os erros que podem e devem ser detectados em tempo de compilação e aqueles que ocorrem em tempo de execução.

Java é relativamente uma linguagem de alto nível, de forma que detalhes de representação da máquina não estão disponíveis através da linguagem. Ela inclui gerenciamento automático de armazenamento, normalmente usando o Garbage Colletor, que ajuda a evitar problemas de segurança.
